# 嵌套函数中的 this 不会从外层函数中继承

  1. 将 this 保存为一个变量，如 that，利用变量的作用域机制传递给嵌套函数（闭包或函数参数）
  2. 继续使用 this，但是把普通函数改为箭头函数，因为箭头函数没有自己的执行上下文，所以它会结成调用函数的 this
