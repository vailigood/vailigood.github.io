# JavaScript 数据类型

JavaScript 是一门弱类型动态语言，这意味这你不需要在声明变量时指定类型，并且遇到类型问题时 JavaScript 也会进行隐式转换。

# 原始类型

原始类型直接存储在执行栈中，闭包会被存入堆中。

  1. Boolean
  2. Null
  3. Undefined
  4. Number
  5. BigInt
  6. String
  7. Symbol

# 引用类型

在 JavaScript 中，Object 可以被看作原始类型的集合。引用类型只会在栈中存放引用地址的指针，具体内容存放在堆中。
